== Description ==
This plugin enables creation of email campaigns via CPT, uploads CSV/XLSX lists, schedules sends, and tracks metrics.

== Installation ==
1. Upload plugin to `/wp-content/plugins/`
2. Activate in WP Admin
3. Configure SendGrid webhook to `/wp-json/email-campaign/v1/webhook/bounce`
4. Create a new Email Campaign
